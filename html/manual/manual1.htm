1. Requirements
 

Full usage of YPL+ requires the following settings. Depending if you are uploading images or just browse through the databases you have different minimum requirenments.

  1.1 Browser
 

For uploading images: Netscape 4.7, MIE 4.0 or equivalent. JavaScipt enabled; I/Frames enabled; Java 1.5
For browsing through the databases: Netscape 4.7, MIE 4.0 or equivalent. JavaScipt enabled; I/Frames enabled;

  1.2 System
 

All OS providing one of the above listed browsers. Monitor resolution should be 1024x728 or higher.
